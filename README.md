## Checklist de SEO & Performance (o que já foi incluído e o que sugerimos):
- ✅ HTML semântico (header/main/footer/section/article)
- ✅ Meta description, og tags, twitter card
- ✅ Mobile-first CSS
- ✅ Links CTA visíveis e acessíveis
- ✅ Remoção de valores/preços (apenas planos descritivos + CTA)
- ✅ Lazy load nas imagens (use `loading="lazy"` nas imagens adicionais)
- ✅ Minimização recomendada (use build step para minificar CSS/JS)
- ✅ Structured data (WebSite + você pode adicionar FAQ schema)
